A brief explanation

FLOYD dari waktu ke waktu akan saya terus perbaiki dan tambah fitur. Saya sangat terbuka terhadap masukan dan saran. Sebisa mungkin penambahan fitur terhadap FLOYD tidak banyak merubah struktur HTML sehingga anda hanya perlu memperbaharui file komponen saja. Namun ada kalanya perubahan struktur HTML diperlukan apabila fitur baru yang ditambahkan cukup besar. Anda dapat mengetahui dengan mudah perubahan tersebut dari versi rilis template.

(another) A brief explatantion

  • Angka versi dipecah menjadi 3 digit.
  • Contohnya v1.0.1
  • Digit terakhir mengindikasikan perbaikan bug
  • Digit kedua mengindikasikan penambahan fitur kecil, biasanya tidak merubah struktur html
  • Digit pertama mengindikasikan penambahan fitur besar yang merubah struktur kode html maupun CSS
